Call for Papers

The International Conference on Intelligent Computing and Communication Technologies (ICICCT-2019) is the premier interdisciplinary platform for the presentation of new advances and research results in the fields of Intelligent Common Computing. The conference will bring together leading academic scientists, researchers and scholars in the domain of interest from around the world.

Topics of interest for submission include, but are not limited to:

   Artificial Intelligen & Machine Learning    Data Mining & Analytics    Computer Vision    Networks and Security    Intelligent Computing
   Evolutionary Algorithms    Databases    Image Processing    Distributed Computing    Cloud Computing
   Statastical learning    Information Retrieval    Morphological Image Processing    Cluster Computing    Intelligent Grid
   Neural Networks    Knowledge Discovery    Image Retrieval    Ubiqutous Computing    Internet of Things
   Machine Learning    Big Data    Document Image Processing    High Performance Computing    Hybrid Systems
   Deep Learning    Text Analytics    Medical Image Processing    Parallel Computing    E-governance, E-commerce, E-Learning
   Natural Language Processing    Web Analytics    Satellite Imagery    Vehicular Networks    Bio-Informatics
   Fuzzy Systems & Genetic Algorithms    Business Intelligence, Data Lakes    Robotics Process Automation    Wireless Sensor Networks    Recommendation System
   Rough Sets    Block Chain    Biometrics    Storage Area Networks    Soft & Fog Computing
   Web Intelligence & Multimedia    HCI Challenges for Data Analytics    Video Processing & Analysis    Multi Agent Systems    Quantum Computing
   Speech Processing    Data Mining & Knowledge Engineering    Pattern Recognition    Information Security    Telemedicine
   Cognitive Science    Graph Mining    Remote Sensing    Cyber Security    Wearable Computing
   Virtual Reality    Video Mining    Stochastic Modeling    Digital Forencis    Green Computing
